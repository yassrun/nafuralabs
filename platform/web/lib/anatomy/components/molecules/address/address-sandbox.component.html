Usage Examples

Basic Form

<nf-address-form
  [value]="address"
  [mode]="'full'"
  [required]="{ line1: true, city: true, countryCode: true }"
  (valueChange)="onAddressChange($event)"
  (validityChange)="onValidityChange($event)">
</nf-address-form>

Read-only View

<nf-address-view
  [value]="address"
  [mode]="'full'"
  [countryLabelResolver]="getCountryLabel">
</nf-address-view>

Compact Checkout Form

<nf-address-form
  [mode]="'compact'"
  [required]="{ line1: true, city: true, countryCode: true }"
  (valueChange)="onShippingAddressChange($event)">
</nf-address-form>

Features

  • ✓ Full and compact display modes
  • ✓ Configurable required fields
  • ✓ Real-time validation with error messages
  • ✓ Debounced value emission (200ms)
  • ✓ Design token-based styling
  • ✓ i18n-ready with translation keys
  • ✓ WCAG accessible (ARIA labels, keyboard navigation)
  • ✓ Reactive forms integration
  • ✓ Domain-agnostic and reusable
  • ✓ OnPush change detection strategy
