# Web workspace (Sektor)

Point d’entrée Angular pour **Sektor BTP**. Le code métier ERP n’est **pas** ici.

| Rôle | Emplacement |
|------|-------------|
| Bootstrap (`main.ts`, `app.config`) | `web/app/`, `web/src/` |
| Platform UI / shell | `../platform/web/` → alias `@platform/*` |
| Pages & services ERP | `../products/sektor-btp/web/app/` → alias `@applications/*` |

Guide complet : [docs/README.md](../docs/README.md).
