## Task 9.3 — Heures supplémentaires (M-RH-03) **P1**

**Barèmes MA (Code du Travail) :**
- HS25 : au-delà 44h/sem ou 8h/jour ouvré → +25 %
- HS50 : entre 21h et 6h (nuit) ou dimanche → +50 %
- HS100 : jour férié → +100 %

**Modèle** :

```ts
export interface HeureSupplementaire {
  id: string;
  employeId: string;
  chantierId: string;
  date: string;
  heuresHS25: number;
  heuresHS50: number;
  heuresHS100: number;
  motif?: string;
  validePar?: string;            // chef chantier
  status: 'SAISIE' | 'VALIDEE' | 'INTEGREE_PAIE' | 'REJETEE';
}
```

**Action** :
1. Saisie depuis pointage chef chantier (cf M-RH-01)
2. Validation hiérarchique
3. Intégration paie automatique (PaieEngine étendu)
4. Calcul brut avec majorations

**Acceptance criteria** :
- [ ] CRUD HS
- [ ] Calcul auto HS25/50/100 selon date + heure
- [ ] Test unitaire `paie-engine.service.spec.ts` cas HS
- [ ] Intégration paie + DAMANCOM
